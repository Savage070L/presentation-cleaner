# Watermark Remover Web (GitHub Pages)

Веб-версия удаления водяных знаков для PDF и PPTX с обработкой в браузере (без сервера).

## Что уже реализовано

- Загрузка `.pdf` и `.pptx`.
- Обработка в браузере через `opencv.js` (`inpaint` по прямоугольной области).
- Экспорт обработанных файлов:
  - PDF -> новый PDF (`_clean.pdf`).
  - PPTX -> новый PPTX (`_clean.pptx`).
- Область NotebookLM по умолчанию + опциональная кастомная область.

## Как запустить локально

Из папки `watermark-web`:

```bash
python3 -m http.server 8080
```

Открыть: `http://localhost:8080`

## Как выложить на GitHub Pages

1. Создайте репозиторий и загрузите содержимое папки `watermark-web`.
2. В репозитории зайдите в `Settings -> Pages`.
3. В `Build and deployment` выберите:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main` (или `master`), папка `/ (root)`
4. Подождите публикацию и откройте URL Pages.

## Текущие ограничения MVP

- PDF пересобирается из изображений страниц (теряется текстовый слой и векторная графика).
- PPTX-обработка идет по файлам `ppt/media/*`; это покрывает типичный кейс водяного знака на изображениях, но не все формы/вектор.
- На очень больших файлах возможны лаги или ограничение памяти браузера.
